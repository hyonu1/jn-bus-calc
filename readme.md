## 📌 프로젝트 개요
* **목적**: 전라남도교육청 소속 교육행정직 공무원의 여비 산출 업무 효율화.
* **주요 기능**: 관내/관외 여비 구분 및 공무원 여비 규정에 따른 자동 계산.
* **설계 원칙**: 법령에 근거한 정확성 및 시스템의 내구성과 품질 최우선.

## 🛠 기술 스택 및 환경
* **언어**: HTML5, CSS3, JavaScript (Vanilla JS).
* **배포**: GitHub Pages.
